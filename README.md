이온트랩 실험을 위한 파이썬기반 프로그램입니다.
실험에 필요한 장비들의 조작 및 실험 진행을 동시다발적으로 진행하기 위해 프로그래밍 되었습니다.
위해 공유 가능한 실험 장비를 여러 실험 셋업에서 활용하기 공용 장비는 서버 컴퓨터에서 컨트롤하고, 클라이언트들이 접근하여 사용하는 방식으로 설계되었습니다.
